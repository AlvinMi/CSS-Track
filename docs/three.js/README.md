## 基本的图形元素

three.js 基本图形元素，都定义在 `[Objects](http://threejs.outsidelook.cn/r89/source/docs/index.html#Reference/Core/Object3D)` 下， 分为 4 个小类:

### 1.点、线、面

- Points: 点或点云表示图形
- Line/LineSegments：直线和虚线，可以使用线或线团表示图形
